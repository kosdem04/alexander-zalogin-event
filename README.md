# AlexHost Site

Локальный frontend-проект на `React + Vite` для многостраничного сайта ведущего мероприятий.

## Что внутри

- `src/pages` — отдельные страницы под каждый формат
- `src/components` — переиспользуемые блоки: `Header`, `Footer`, `CTASection`, `LeadForm`, `SectionTitle`
- `src/router` — маршрутизация проекта
- `src/assets` — локальные брендовые ассеты
- `src/styles` — глобальные стили

## Маршруты

- `/` — главная
- `/weddings` — свадьбы
- `/corporate` — корпоративы
- `/anniversaries` — дни рождения / юбилеи
- `/graduations` — выпускные
- `/gala-dinners` — гала-ужины / клиентские вечера
- `/ceremony` — выездная церемония
- `/mass-events` — массовые мероприятия
- `/contacts` — контакты / заявка

## Запуск локально

1. Установить зависимости:

```bash
npm install
```

2. Запустить dev-сервер:

```bash
npm run dev
```

3. Открыть адрес, который покажет Vite в терминале. Обычно это `http://localhost:5173`.

## Сборка

```bash
npm run build
```
